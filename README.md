# Project Automation Login Page OrangeHRM - Quiz 3

## Catatan:
Untuk folder "node_modules" sengaja saya masukkan ke .gitignore mas karena ukuran yang terlalu besar.

## Cara Menjalankan Project:
1. Clone repository ini.
2. Buka terminal di folder project.
3. Jalankan perintah: `npm install` (untuk mengunduh library yang diperlukan dan menampilkan folder node_modules lagi).
4. Jalankan Cypress dengan: `npx cypress open` atau `npx cypress run`.
